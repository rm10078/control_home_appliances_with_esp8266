## üîåÔ∏è Connection 

- L0 = GPIO16 OR D0 (TUBE_1)
- L1 = GPIO12 OR D6 (TUBE_2)
- L2 = GPIO05 OR D1 (OTHERS)
- FAN = GPIO13 OR D7
- IR sensor = GPIO14 = D5
- DATA LED = GPIO02 = D4
